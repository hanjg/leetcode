## 位操作 ##
### 与 ###
```txt
x & 0s = 0
x & 1s = x
x & x = x
```

#### n&(n-1) ####
> 最低位的1置0。
- [2 的幂次方](../src/bitManipulation/PowerofTwo.java)

#### n&(-n) ####
> 保留最低位的1，其余全部置0

#### 取消值 ####
- x&0=0

### 或 ###
```txt
x | 0s = x
x | 1s = 1s
x | x = x
```
#### 设置值 ####
- x|1=x

### 异或 ###
```txt
x ^ 0s = x
x ^ 1s = ~x
x ^ x = 0
```

- 判断两个数是否异号。
```txt
	(x ^ y) < 0 //异号
```

- 去除重复两次的数。
  - [只出现一次的数](../src/bitManipulation/SingleNumberIII.java)

- 交换两个数。慎用。
> a,b指向同一个地址时失效
```txt
	a ^= b;
	b ^= a;
	a ^= b;
```
