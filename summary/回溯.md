## 回溯 ##
- 特殊的DFS

### 求所有解 ###
```
void backTracking(选择列表，选择路径) {
    if (满足条件) {
        result.add(结果);
        return;
    }
    for (选择：选择列表) {
        路径+选择；
        backTracking(选择列表，选择路径);
        路径-选择；
    }
}
```

### 求一个解 ###
```
Object backTracking(选择列表，选择路径) {
    if (满足条件) {
        return 结果；
    }
    for (选择：选择列表) {
        路径+选择；
        Object target = backTracking(选择列表，选择路径);
		if (target != null){
			return 结果；
		}  
        路径-选择；
    }
	return null;
}
```
